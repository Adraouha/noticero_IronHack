# Portafolio web de Hassan Adraou

Proyecto front-end creado con HTML y CSS para presentar mi perfil profesional, experiencia laboral, educacion, intereses personales y un catalogo de proyectos publicados en GitHub Pages.

## Objetivo

Crear una web personal que funcione como portafolio real: presentacion, habilidades, experiencia profesional, proyectos destacados, enlaces a GitHub, animaciones hover, diseno responsive y formulario de contacto simple.

## Caracteristicas

- Estructura semantica con `header`, `nav`, `main`, `section`, `article`, `form` y `footer`.
- Maquetacion con Flexbox en cabecera, tarjetas, proyectos, preferencias, intereses y contacto.
- Estilo visual unificado en todas las paginas: `index.html`, `preferencias.html` y `tarjeta.html`.
- Menu de navegacion sin iconos, con los mismos colores y comportamiento hover en todo el sitio.
- Boton para descargar el CV en PDF desde el menu y la portada.
- Seccion de proyectos con enlaces a demos de GitHub Pages y repositorios.
- Seccion de experiencia laboral basada en el CV online.
- Seccion de educacion y proyectos profesionales.
- Pagina de preferencias con musica basada en el proyecto Hack Mi Musica.
- Tarjeta personal con bio mejorada, intereses y audio de Spotify.
- Animaciones hover en navegacion, botones y tarjetas de proyecto.
- Animacion con `@keyframes` en una tarjeta destacada.
- Media queries para adaptar la web a formato vertical en pantallas pequenas.
- Formulario de contacto simple con campos de nombre, email y mensaje.
